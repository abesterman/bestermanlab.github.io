How to Use

  1. Click on the "Access Data Repository" link above.
  2. Navigate through the available projects in the GitHub repository.
  3. Download the data files and associated code for the project you're interested in.
  4. Refer to the README file in each project folder for specific usage instructions.

If you have any questions about the data or code, please open an issue on our GitHub repository or contact us directly at abesterman@health.ucsd.edu.
